Quick Web Accessibility Implementation with All In One Accessibility!

The Odoo All in One Accessibility Widget is developed based on AI and assistive technology that helps to enhance website accessibility. It improves the accessibility of your website to people with hearing or vision impairments, motor impaired, color blind, dyslexia, cognitive & learning impairments, seizure and epileptic, and ADHD problems. This interface allows users to select the accessibility features as per their needs and peruse the content.
